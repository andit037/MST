# MST
How To Run:
1. Install nodejs
2. Install appium : npm install -g appium
3. git clone
4. set the appium directory on Katalon Studio Preferences
6. Go to test-suite on this katalon project
7. Play or Run with Emulator Android (Android SDK installed)
